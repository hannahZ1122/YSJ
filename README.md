# AI-Assisted Probiotic Recommendation System

## Project Overview
This project presents an AI-assisted probiotic recommendation module developed as a WeChat Mini Program. The system combines a rule-based decision engine with a large language model (LLM) to generate explainable, consistent, and personalized recommendations for consumers.

## Problem
Consumers lack structured knowledge and struggle to select appropriate probiotic products based on their individual conditions, leading to low confidence and suboptimal decision-making.

## Solution
A hybrid architecture is adopted:
- Rule-based system → ensures consistency, reliability, and control over recommendations  
- LLM → generates natural-language explanations to improve user understanding and trust  

## System Architecture
The system follows a hybrid decision-support design:
- Input Layer: Structured questionnaire and optional report upload  
- Decision Layer: Rule-based recommendation engine  
- Explanation Layer: LLM-generated personalized explanations  
- Interface Layer: WeChat Mini Program for end-to-end user interaction  

## Key Features
- Structured questionnaire input  
- Deterministic rule-based recommendation engine  
- AI-generated explanations (LLM)  
- End-to-end Mini Program workflow
