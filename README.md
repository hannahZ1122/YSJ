# AI-Assisted Probiotic Recommendation System

## Project Overview
This project presents an AI-assisted probiotic recommendation module developed as a WeChat Mini Program. The system combines a rule-based decision engine with a large language model (LLM) to generate explainable and personalized recommendations.

## Problem
Consumers lack structured knowledge and struggle to select appropriate probiotic products based on their individual conditions.

## Solution
A hybrid architecture:
- Rule-based system → ensures consistency and control
- LLM → generates natural-language explanations

## Key Features
- Structured questionnaire input
- Rule-based recommendation engine
- AI-generated explanations
- End-to-end Mini Program workflow
